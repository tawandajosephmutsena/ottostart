Apex Finance approached us with a platform suffering from significant legacy debt. Their user interface hadn't been updated in six years, resulting in a disjointed experience that was driving new users away.

The Challenge

Users were forced to navigate three different sub-domains to complete a single international transfer. The friction was palpable, with a 65% drop-off rate at the "Verification" step.

  • High churn rate during onboarding
  • Inconsistent visual language

Mobile Native

iOS & Android / React Native

Design System

We created "Apex UI", a comprehensive library ensuring consistency across web and mobile products.

Accessible

WCAG 2.1 AA Compliant color contrast and navigation patterns.

The Solution

Unified
Dashboard

We collapsed the fragmented architecture into a Single Page Application (SPA). A persistent sidebar navigation gives users instant access to all tools.

Real-time Data

WebSocket integration for live market updates.

Smart Drawers

Contextual actions without leaving the main view.
