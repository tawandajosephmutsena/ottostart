The
Brief

Apex Finance was suffering from significant legacy debt. Their interface hadn't been updated in six years, resulting in a disjointed experience driving new users away.

Our goal was twofold: modernize the visual language to compete with neobank startups, while preserving the dense, data-rich utility that their power users relied on daily.

We needed to bridge the gap between institutional trust and modern usability, creating a system that felt both secure and cutting-edge.

Mobile Experience

Performance First

WebGL rendering reduced load times by 400% for heavy data sets.

The Challenge

Fragmented
Ecosystems

Users were navigating three sub-domains for one transfer. A 65% drop-off rate at the "Verification" step signaled a critical failure in the UX flow.

  • High churn during onboarding
  • Inconsistent visual language
Wireframe Phase
The Solution

Unified Dashboard
Architecture

We collapsed the fragmented architecture into a SPA. A persistent sidebar navigation gives users instant access to all tools, while contextual drawers handle complex tasks.
