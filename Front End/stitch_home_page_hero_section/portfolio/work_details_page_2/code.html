The Brief

Apex Finance approached us with a platform suffering from significant legacy debt. Their user interface hadn't been updated in six years, resulting in a disjointed experience that was driving new users away during the onboarding process.

Our goal was twofold: modernize the visual language to compete with neobank startups, while preserving the dense, data-rich utility that their power users relied on daily. We needed to bridge the gap between complexity and usability.

Mobile Interface
400% Faster Load Times
The Challenge

Fragmented Systems & User Friction

Users were forced to navigate three different sub-domains to complete a single international transfer. The friction was palpable, with a 65% drop-off rate at the "Verification" step. The legacy backend required a complete overhaul of the frontend architecture to support real-time data without page reloads.

  • High churn rate during onboarding
  • Inconsistent visual language across tools
  • Slow rendering of market data (WebGL issues)

Initial wireframe sketches

The Solution

Unified Dashboard Architecture

We collapsed the fragmented architecture into a Single Page Application (SPA). A persistent sidebar navigation gives users instant access to all tools, while contextual drawers handle complex tasks without leaving the main view.
